# 🚀 React TypeScript Projects

## 📌 Descripción
Colección de proyectos frontend desarrollados con **React, TypeScript y Next.js**, enfocados en aplicar conceptos clave, herramientas modernas y resolver casos del mundo real.

Este repositorio forma parte de mi proceso de aprendizaje mediante el curso **"React y TypeScript - La Guía Completa Creando +10 Proyectos"** impartido por **Juan Pablo De La Torre Valdez**.

> ⚠️ Nota: Los proyectos Full Stack se encuentran en un repositorio separado.

---

## 🎯 Objetivo
Desarrollar y consolidar habilidades en frontend moderno mediante la creación de proyectos prácticos, aplicando buenas prácticas y herramientas actuales.

---

## 🧠 Temas aprendidos
- ⚛️ Componentes y Props  
- 🔄 State y Hooks (useState, useEffect, custom hooks)  
- 🧩 useReducer para manejo de estado complejo  
- 🌐 Consumo de APIs  
- 📦 Estado global (Context API, Zustand)  
- 🧪 Validación de formularios (React Hook Form, Zod)  
- 🛣️ Routing con React Router DOM  
- ⚡ Next.js (App Router, Server Actions)  
- 🗄️ Prisma ORM (modelado y acceso a base de datos)  
- 🧱 Arquitectura y organización de proyectos  

---

## 📂 Proyectos incluidos
- 🛒 Carrito de Compras  
- 💰 Calculadora de Propinas  
- 🏋️ Contador de Calorías  
- 🔄 Migración a useReducer

- 🌦️ Aplicación de Clima  
- 💲 Cotizador de Criptomonedas  
- 🍽️ Buscador de Recetas  
- 🍔 Quiosco de Comida (Next.js + Prisma)
